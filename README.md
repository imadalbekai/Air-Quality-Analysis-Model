# Environmental Sensor Regression

Predicting **temperature**, **relative humidity**, and **absolute humidity** in Italy using **chemical sensor data** and **time-of-day features**. This project demonstrates regression modeling and feature analysis on real-world environmental data.

## Dataset
- Sensor readings of gases (CO, NO₂, etc.)  
- Time of day  
- Targets: Temperature (°C), Relative Humidity (%), Absolute Humidity (g/m³)

## Approach
1. Data preprocessing (handling missing values, scaling, encoding time features)  
2. Regression models: Linear Regression, Random Forest, Gradient Boosting, XGBoost  
3. Evaluation with MAE, MSE, R²  
4. Feature importance analysis

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, XGBoost, randomforest... Matplotlib, Seaborn, Jupyter Notebook
